This package contains several demo assets that are in the public domain under a Creative Commons 0 licence CC0. You can find them in the Shared folder clearly marked with Public Domain next to their folder names and with their respective licenses inside. 

Everything else in this package is distributed under the standard Unity Asset Store license

To use this package with the new SRP Lightweight Pipeline, please unpack the LighteightContents unity package and then delete the "Standard Version" folder after you have successfully set up your project for the Lightweight workflow. Then, follow the instructions in the manual to learn how to set up cameras and the component itself to work with SRPs.
